# 欢迎来到我的博客 / Welcome to My Blog

这是我的第一篇博客文章！在这里，我会分享我的学习心得、技术笔记和生活随笔。

This is my first blog article! Here I share my learning notes, tech articles, and thoughts.

## 博客功能 / Blog Features

本博客支持以下功能 / This blog supports:

- **Markdown 语法 / Markdown syntax**: 支持完整的 Markdown 语法 / Full Markdown syntax support
- **数学公式 / Math formulas**: 支持 LaTeX 数学公式渲染 / LaTeX formula rendering
- **分类浏览 / Category filter**: 可以按分类筛选文章 / Filter articles by category
- **标签搜索 / Tag search**: 可以搜索特定标签的文章 / Search articles by tags
- **排序功能 / Sorting**: 支持按日期和标题排序 / Sort by date or title

## 使用方法 / How to Use

### 浏览文章 / Browse Articles

1. 在左侧边栏选择分类 / Select category in the left sidebar
2. 输入标签名称进行搜索 / Search by tag name
3. 点击文章卡片阅读全文 / Click article card to read more

### 添加新文章 / Add New Article

1. 在 `contents/blog/` 目录下创建新的 `.md` 文件 / Create new `.md` file in `contents/blog/`
2. 在 `articles.yml` 中添加文章元数据 / Add article metadata in `articles.yml`
